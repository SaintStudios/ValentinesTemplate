# Landing Page Images

Place your landing page images in this folder for use in components.

## Image Guidelines

- Use optimized formats (WebP, PNG, JPG)
- Consider mobile responsiveness when sizing images
- Recommended max width: 1920px for hero/background images
- Keep file sizes under 500KB where possible
